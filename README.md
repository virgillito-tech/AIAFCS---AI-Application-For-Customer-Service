# AIAFCS---AI-Application-For-Customer-Service
Una piattaforma full-stack basata su GenAI e RAG (Retrieval-Augmented Generation) per potenziare gli operatori di Customer Service. Il sistema ascolta e trascrive le chiamate in tempo reale, suggerendo soluzioni tecniche basate sulla documentazione aziendale e automatizzando il data-entry nel CRM.
